# NeoForge 1.21.1

Đây là bản NeoForge 1.21.1 gốc, được giữ nguyên phần source và cấu hình build.

Yêu cầu: JDK 21.

Build:

```bat
gradlew.bat clean build
```

Chạy môi trường phát triển:

```bat
gradlew.bat runClient
```

Theo `build.gradle` gốc, JAR release được đưa vào thư mục `../mods` của workspace.
